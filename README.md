# ☕ Coffee Shop Voting App

A full-stack web application where users can register, login, and vote for their favorite coffee drinks. Built with React, Flask, and SQLite.

## 🚀 Live Demo

- **Frontend:** https://the-great-coffee-1.onrender.com

## 📸 Features

- 🔐 **User Authentication** - Register, Login, Logout with secure password hashing
- ☕ **Coffee Menu** - Browse available coffee drinks with descriptions and prices
- ❤️ **Voting System** - Vote for your favorite coffee (toggle on/off)
- ⭐ **Rating System** - Dynamic rating updates based on votes
- 📊 **Real-time Updates** - Vote counts update instantly
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile
- 🎨 **Clean UI** - Modern and intuitive interface
- 🔒 **Secure Sessions** - Session-based authentication with cookies

## 🛠️ Tech Stack

### Frontend
- **React 18** - UI framework
- **Vite** - Build tool
- **React Router DOM** - Navigation and routing
- **Axios** - HTTP client for API calls
- **CSS3** - Custom styling

### Backend
- **Flask** - Python web framework
- **Flask-CORS** - CORS handling
- **Flask-SQLAlchemy** - ORM for database
- **SQLite** - Database (development)
- **bcrypt** - Password hashing
- **Gunicorn** - Production WSGI server
